# Form App

Registration form with input fields, validation, and feedback.

## Run

```bash
cargo run
```

## Controls

- `Tab` — Switch between fields
- `Enter` — Submit form
- `Backspace` — Delete character
- Type normally to enter text
- `q` — Quit

## What's included

- Multiple `Signal<String>` fields for form state
- Field switching with active indicator
- Client-side validation (required name, email format)
- Success/error feedback messages
- Inline CSS with status color classes
